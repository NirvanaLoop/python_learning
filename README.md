# Python Course 项目说明

本项目包含多个Python教学案例与课程资料，适合初学者和进阶学习者。

## 目录结构

- `贪吃蛇/`：模块化贪吃蛇游戏项目，含AI自动控制与详细教学教案。
- `五子棋教学.ipynb`：五子棋项目教学。
- `python/`：其他Python学习资料与代码。
- `Lesson*.ipynb`/`Lesson*.pdf`：配套课程讲义与练习。

## 贪吃蛇项目特色

- **模块化设计**：蛇、食物、AI、主循环等均为独立模块，便于扩展和维护。
- **AI模式**：支持自动寻路吃食物，体验算法乐趣。
- **详细教案**：引导学生拆解问题、模块化思考，适合课堂教学。
- **可自定义胜利分数、模式切换**。

## 快速开始

1. 进入 `贪吃蛇/` 目录，运行 `snake_game.py`。
2. 按提示选择AI或手动模式，输入胜利分数。
3. 参考 `贪吃蛇_AI教学引导教案.ipynb` 进行教学或自学。
