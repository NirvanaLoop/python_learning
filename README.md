# 🐍 Python 编程课程全集

> 一个全面的Python编程教学项目，从零基础到人工智能应用，涵盖游戏开发、数据结构、面向对象编程及大模型微调等前沿技术。

## 📚 项目概述

本项目是一个系统性的Python编程教学资源库，适合不同层次的学习者：

- **初学者**：从基础语法到实践项目
- **进阶者**：游戏开发、算法应用、AI编程
- **教育者**：完整的教学教案和课程设计

## 🗂️ 项目结构

### 🎮 游戏开发项目

#### 贪吃蛇游戏 (`贪吃蛇/`)

**⭐ 明星项目** - 完整的模块化游戏开发教学案例

- 🏗️ **模块化架构**：蛇类、食物类、游戏引擎、AI控制器
- 🤖 **智能AI模式**：
  - 简单贪心算法（直线靠近食物）
  - BFS路径规划（安全寻路）
- 📖 **详细教学文档**：step-by-step实现指南
- ⚙️ **功能特色**：
  - 玩家/AI模式切换
  - 自定义胜利分数
  - 实时游戏状态显示
  - 碰撞检测和游戏逻辑

```bash
# 快速体验
cd 贪吃蛇/
python snake_game.py
```

#### 五子棋游戏 (`五子棋/`)

- 📱 **多版本实现**：
  - `五子棋教学.ipynb`：基础教学版本
  - `五子棋弹窗版.ipynb`：图形界面版本
  - `五子棋ipycanvas版.ipynb`：高级交互版本
- 🎯 **教学重点**：GUI编程、游戏逻辑、胜负判断

### 📖 Python基础课程 (`python/`)

#### 系统化课程讲义

- **Lesson1-6**: 完整的Python基础教程（.ipynb + .pdf）
  - 基础语法与数据类型
  - 控制结构与函数
  - 数据结构与算法
  - 文件操作与异常处理
  - 模块化编程
  - 面向对象编程

#### 面向对象编程专题

- `面向对象基础教程（一）.ipynb`：类与对象基础
- `面向对象基础教程（二）.ipynb`：继承、多态与封装

#### 深度学习资源

- `数据结构与算法.pdf`：计算机科学基础
- **完整Python教程库**：从入门到进阶的markdown教程集

### 🚀 大模型与AI (`大模型/`)

#### 大模型微调技术课程

- 📋 **完整课程大纲**：从理论到实践的全面指南
- 🔧 **技术涵盖**：
  - LoRA、Adapter、Prompt-based微调
  - 分布式训练与优化
  - 数据工程与评估体系
  - 实战项目案例
- 📊 **实用资源**：
  - 理论PDF文档集
  - Cursor开发工具教程
  - DeepSeek实践指南
  - Transformer架构解析

#### AI应用开发

- 🤖 **智能体开发**：从理解到实操
- 💡 **Prompt工程**：提示词设计与优化
- 🛠️ **零代码AI应用**：小白友好的开发指南


## 🚀 快速开始

### 环境要求

```bash
Python 3.7+
pip install pygame  # 游戏开发
pip install jupyter # Notebook支持
pip install numpy pandas matplotlib  # 数据分析
```

### 学习路径建议

#### 🥇 零基础入门

1. **Python基础** → `python/Lesson1-6`

3. **进阶应用** → `五子棋/` GUI编程
