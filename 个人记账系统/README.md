# 💰 个人记账系统

一个简单实用的个人记账系统，适合Python初学者学习项目开发。

## 📚 项目简介

这是一个基于Python开发的命令行记账系统，可以帮助用户记录日常的收入和支出，并提供简单的统计功能。

### 功能特点

- ✅ 记录收入和支出
- ✅ 按类别分类管理
- ✅ 查看交易记录
- ✅ 账户余额统计
- ✅ 支出分类统计
- ✅ 数据持久化（JSON文件存储）

## 🏗️ 项目结构

```
个人记账系统/
├── main.py              # 主程序入口
├── account.py           # 账户管理类
├── transaction.py       # 交易记录类
├── web_app.py           # Flask Web界面入口
├── templates/
│   └── index.html       # Web模板
├── static/
│   └── style.css        # 简单样式
├── requirements.txt     # 依赖文件
├── README.md           # 项目说明
├── 教学文档.md         # 教学指南
├── 使用示例.md         # 使用示例
└── data/               # 数据存储目录
    └── transactions.json # 交易记录文件（自动生成）
```

## 🚀 快速开始（命令行）

### 1. 运行程序

```bash
python main.py
```

### 2. 使用说明

程序启动后会显示主菜单：

```
==================================================
个人记账系统
==================================================
1. 添加收入
2. 添加支出
3. 查看所有记录
4. 查看收入记录
5. 查看支出记录
6. 按类别查询
7. 查看账户摘要
0. 退出
==================================================
```

## 🌐 Web界面（Flask）

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 启动Web服务
```bash
python web_app.py
```

默认访问地址：
```
http://127.0.0.1:5000/
```

### 3. Web功能
- 添加收入/支出记录（表单）
- 查看交易列表
- 查看总收入、总支出、余额
- 查看支出分类统计

> Web界面和命令行共用同一个数据文件 `data/transactions.json`

## 📖 代码结构说明

### Transaction 类 (`transaction.py`)

表示单笔交易记录，包含：
- 金额 (amount)
- 类别 (category)
- 描述 (description)
- 类型 (transaction_type: "收入" 或 "支出")
- 日期 (date)

### Account 类 (`account.py`)

账户管理类，负责：
- 管理所有交易记录
- 数据的保存和加载
- 提供查询和统计功能

### main.py

主程序，提供命令行交互界面。
